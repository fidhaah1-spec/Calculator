# Calculator

A simple calculator application built with HTML, CSS, and JavaScript.

## Features

- Basic arithmetic operations: addition, subtraction, multiplication, division, and percentage.
- "Clear Entry" (CE) and "Clear" (C) functionalities.
- "Backspace" functionality to correct mistakes.
- Keyboard support for numbers, operators, "Enter", "Backspace", and "Delete".
- Modern and visually appealing color scheme.
- Accessible with `aria-label` attributes for screen readers.
- Prevents multiple operators in a row.
